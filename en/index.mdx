## Welcome to Ruoom Core

Ruoom Core is the open source foundation of our project, designed to provide a robust and flexible framework for building powerful applications. It serves as the heart of our ecosystem, enabling seamless integration with a variety of plugins to extend its functionality.

### Key Features

- **Modular Architecture**: Ruoom Core's modular design allows developers to easily customize and extend the framework with plugins.
- **Open Source**: As an open source project, Ruoom Core encourages community contributions and collaboration.
- **Extensible with Ruoom Plugins**: Ruoom's open source core can be extended with a wide range of plugins published by Ruoom, allowing for enhanced functionality and customization.

### Getting Started

To start using Ruoom Core, follow the [quickstart guide](en/quickstart) to set up your development environment. Whether you're developing a new plugin or integrating with existing ones, Ruoom Core offers the tools and flexibility you need to succeed.

For more detailed information on individual plugins, please refer to the specific sections in this documentation.
