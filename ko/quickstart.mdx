## Ruoom Core 빠른 시작 가이드

Ruoom Core에 오신 것을 환영합니다! 개발 환경을 설정하고 Ruoom Core를 사용하려면 다음 단계를 따르십시오.

### 환경 설정

1. **리포지토리 복제**: GitHub에서 Ruoom Core 리포지토리를 로컬 컴퓨터로 복제합니다.
   
   ```bash
   git clone https://github.com/your-username/ruoom-core.git
   cd ruoom-core
   ```

2. **데이터베이스 초기화**: "Ruoom"이라는 데이터베이스, "ruoom_admin"이라는 사용자 및 "password"라는 암호로 PostgreSQL을 설정합니다. 그런 다음 다음 명령을 실행하여 마이그레이션을 적용합니다:

   ```bash
   python manage.py migrate
   ```

3. **개발 서버 실행**: 로컬 개발 서버를 시작하여 Ruoom Core를 실행합니다.

   ```bash
   python manage.py runserver
   ```

   이제 `localhost:8000`에서 애플리케이션에 액세스하고 초기 계정을 생성할 수 있습니다.

### 다음 단계

- 이 문서의 API 및 플러그인 개발 가이드를 탐색하여 Ruoom Core의 기능을 확장하십시오.
- GitHub에서 문제를 제출하거나 풀 요청을 제출하여 프로젝트에 기여하십시오.

자세한 지침은 Ruoom Core 리포지토리의 README 파일을 참조하십시오.
