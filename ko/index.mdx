## Ruoom Core에 오신 것을 환영합니다

Ruoom Core는 강력한 애플리케이션을 구축하기 위한 견고하고 유연한 프레임워크를 제공하도록 설계된 프로젝트의 오픈 소스 기반입니다. 이는 다양한 플러그인과의 원활한 통합을 가능하게 하여 생태계의 중심 역할을 합니다.

### 주요 기능

- **모듈식 아키텍처**: Ruoom Core의 모듈식 설계는 개발자가 플러그인을 사용하여 프레임워크를 쉽게 사용자 정의하고 확장할 수 있도록 합니다.
- **오픈 소스**: 오픈 소스 프로젝트로서 Ruoom Core는 커뮤니티 기여와 협업을 장려합니다.
- **확장 가능하고 효율적임**: 성능을 염두에 두고 구축된 Ruoom Core는 확장성과 효율성을 최적화했습니다.

### 시작하기

Ruoom Core를 사용하려면 GitHub 페이지에서 리포지토리를 복제하고 README 파일에 제공된 설정 지침을 따르십시오. 새 플러그인을 개발하든 기존 플러그인과 통합하든 Ruoom Core는 성공하는 데 필요한 도구와 유연성을 제공합니다.

API 및 플러그인 개발에 대한 자세한 정보는 이 문서의 특정 섹션을 참조하십시오.
