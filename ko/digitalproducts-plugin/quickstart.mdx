## Digital Products Plugin 빠른 시작 가이드

Digital Products Plugin은 디지털 제품을 관리하기 위한 강력한 도구입니다. 이 빠른 시작 가이드는 디지털 제품 환경을 설정하고 Digital Products Plugin을 시작하는 과정을 안내합니다.

## 설정

### 전제 조건

시작하기 전에 Ruoom Core 오픈 소스 리포지토리를 [빠른 시작 가이드](/ko/quickstart)를 따라 설치했는지 확인하십시오.

또한, Digital Products Plugin의 전제 조건으로 Payment Plugin이 필요합니다. [빠른 시작 가이드](/ko/payment-plugin/quickstart)를 따라 Payment Plugin을 설치했는지 확인하십시오.

### 단계별 지침

1. **리포지토리 다운로드**

[Ruoom Code Lab](https://ruoomsoftware.com/code-lab/)에서 리포지토리를 구매한 후 제공된 소스 코드 패키지를 다운로드하십시오.

리포지토리를 구매한 후 GitHub에 접근하여 리포지토리를 클론하고 풀 요청을 통해 프로젝트에 기여할 수 있습니다. 이는 커뮤니티의 협력과 개선을 촉진합니다.

2. **플러그인 설치**

다운로드한 리포지토리 코드를 Ruoom Core 프로젝트의 `plugins` 폴더에 배치합니다. 그런 다음, `settings.py` 파일의 `INSTALLED_APPS` 리스트에 `plugins.digitalproducts`를 추가하여 Digital Products Plugin이 Ruoom Core 프레임워크에서 인식되도록 합니다.

3. **마이그레이션 실행**

플러그인을 설치한 후 다음 명령을 실행하여 필요한 데이터베이스 마이그레이션을 적용하십시오:

```bash
python manage.py migrate
```

이 단계는 Digital Products Plugin에서 필요한 모든 데이터베이스 테이블이 생성되고 사용할 준비가 되었는지 확인합니다.

4. **개발 서버 실행**

Digital Products Plugin을 실행하려면 로컬 개발 서버를 시작하십시오.

```bash
python manage.py runserver
```

로그인하면 관리 탐색 바에 "스토어 -> 디지털 제품" 옵션이 표시되어 설치가 성공적으로 완료되었음을 나타냅니다.

5. **디지털 제품 구성**

Digital Products Plugin을 사용하면 디지털 제품을 정의하고 관리할 수 있습니다. "스토어 -> 디지털 제품" 페이지로 이동하여 디지털 제품을 추가하고 구성하기 시작하십시오.

디지털 제품을 추가할 때 제품에 대한 링크와/또는 비밀번호를 지정하고 가격과 설명을 입력합니다. Ruoom은 이후에 고객이 제품을 구매할 수 있는 링크를 자동으로 생성합니다.

## 다음 단계

- GitHub에서 문제를 제출하거나 풀 요청을 제출하여 프로젝트에 기여하십시오.

자세한 지침은 Digital Products Plugin 리포지토리의 README 파일을 참조하십시오.
